Quelques commandes de base !

print()

Sert à afficher sur l'écran aussi bien des valeurs que des phrases.
Il est obligatoirement précédé de paranthèses pour lui indiquer quoi écrire.

>>> print("Hello World !")
Hello World !
>>> print(1+1)
2
>>> print("Salut !",2+2)
Salut ! 4

input()

Comment récupérer une variable demandée à l'utilisateur.
La commande " = " permet d'affecter une valeur, une variable à un valeur.

>>> variable = float(input("float: nb réels"))
>>> variable = str(input())
>>> variable = int(input("int: nb entiers"))
>>> variable = eval(input("eval: chaine de caracteres))
